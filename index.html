o Utilisation de navigateur Web en réseau local ou via Internet.

- Accès via Exe:

o L’extension « .exe » signifie « exécutable » est une application destinée spécifiquement aux systèmes basés sur Windows. Les fichiers .exe permettent également d’installer des programmes avant l’exécution sur l’ordinateur.

- Accès via Apk:

o C'est tout simplement un format de fichier contenant l'ensemble des éléments nécessaires à l'installation d'une application sur Android, Un peu comme un fichier « .exe » sur Windows.

Créer son Entreprise

L’enregistrement peut se faire de plusieurs méthodes :

1. Compléter le formulaire avec des données minimum (adresse mail, numéro de téléphone, mot de passe).

2. Modes de vérification se présentent :

- Par adresse mail : pour cette méthode il faut vérifier si le mail est valide et envoyer un code d’activation à l’adresse mail du propriétaire. Après la vérification, il faut rediriger le client vers la page d’activation de son compte.

- Par SMS : on envoie un code d’activation au numéro de téléphone entré et rediriger le visiteur vers la page d’activation du compte.

3. Compte Google, Facebook, etc. : pour cette méthode on vérifie si sur l’ordinateur du client il y a ses comptes synchronisés (comptes ci-haut cités), on crée le compte par rapport à un des comptes sélectionnés.
